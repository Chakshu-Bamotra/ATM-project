# 🏧 ATM Project

A simple and interactive **ATM Simulation Project** built to demonstrate core banking operations such as balance checking, cash withdrawal, deposits, and account management. This project helps beginners understand programming fundamentals, file handling/database concepts, and basic banking logic.

## 🚀 Features

- 🔐 User Authentication / PIN Verification
- 💰 Balance Inquiry
- 💵 Cash Withdrawal
- 📥 Deposit Money
- 🔄 Transaction Handling
- 📋 Account Information Display
- 🛡️ Input Validation & Error Handling
- 🖥️ Simple Console-Based Interface

## 🛠️ Technologies Used


- **Language:** HTML, CSS, JS, CPP
- **IDE:** VS Code
